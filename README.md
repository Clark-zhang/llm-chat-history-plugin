## 🚀 Quick Start

1. **Install** from VS Code Marketplace - search "AI Chat History Saver"
2. **Chat** with your AI assistants as usual
3. **Done!** Conversations auto-save to `.llm-chat-history/`

### ☁️ Cloud Sync (Optional)

1. Enable: `chatHistory.cloudSync.enabled = true`
2. Click the cloud icon in status bar to login
3. Access your history anywhere via web dashboard!

## ⚙️ Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `chatHistory.autoSave` | `true` | Enable automatic saving |
| `chatHistory.outputDirectory` | `.llm-chat-history/history` | Save location |
| `chatHistory.cloudSync.enabled` | `false` | Enable cloud sync |
