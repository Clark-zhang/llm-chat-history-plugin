# 🔍 搜索功能使用指南

## 功能概述

AI Chat History Saver v0.2.8 新增了强大的全文本搜索功能，让您可以快速找到任何保存的对话内容。

## 如何使用

### 1. 打开搜索界面

1. 在 VS Code 中按 `Ctrl+Shift+P` (Windows/Linux) 或 `Cmd+Shift+P` (macOS) 打开命令面板
2. 输入并选择：`LLMChatHistory: Search Chat History`
3. 或者在扩展面板中找到对应命令

### 2. 输入搜索关键词

- 在搜索框中输入您要查找的关键词
- 支持中英文混合搜索
- 可以搜索对话内容、标题、用户消息、助手回复等

### 3. 查看搜索结果

搜索完成后，会显示匹配的对话列表：
- **对话标题** - 显示匹配的对话标题
- **文件信息** - 显示文件名和匹配次数
- **详细信息** - 显示创建时间和消息数量

### 4. 打开对话文件

- 从结果列表中选择一个对话
- 文件将自动打开，并跳转到第一个匹配的位置
- 匹配的文本行会被高亮显示

## 搜索特性

### 📝 智能匹配
- **标题搜索** - 对话标题匹配权重最高
- **内容搜索** - 搜索用户和助手的所有消息内容
- **工具调用** - 搜索工具使用记录和结果
- **元数据** - 搜索创建时间、会话ID等信息

### 🎯 相关性排序
- 搜索结果按相关性得分排序
- 标题匹配 > 用户/助手消息 > 工具调用 > 元数据
- 精确匹配得分更高

### 🌍 多语言支持
- 支持中英文界面
- 自动适应 VS Code 语言设置
- 搜索结果显示支持双语

## 搜索示例

### 示例 1: 搜索技术关键词
```
搜索: "React authentication"
结果: 显示所有包含 React 认证相关内容的对话
```

### 示例 2: 搜索代码片段
```

结果: 显示所有包含 useState Hook 的对话
```

### 示例 3: 搜索工具调用
```

结果: 显示所有使用 grep 工具的对话
```

### 示例 4: 搜索对话标题
```
搜索: "implementing"

```

## 高级用法

### 组合搜索
```

结果: 显示同时包含 "React"、"component"、"lifecycle" 的对话
```

### 搜索特定类型内容
- 用户消息: 搜索出现在 💬 用户标记下的内容
- 助手回复: 搜索 🤖 助手标记下的内容
- 工具结果: 搜索 🔧 工具使用记录

## 故障排除

### 没有搜索结果
- 检查是否有保存的对话文件 (`.llm-chat-history/history/`)
- 确认搜索关键词拼写正确
- 尝试使用更简短或不同的关键词

### 搜索速度慢
- 大量对话文件时搜索可能需要一些时间
- 这是正常现象，扩展正在处理所有文件

### 文件无法打开
- 确保对话文件没有被删除或移动
- 检查工作区路径是否正确

## 技术实现

### 搜索算法
- **全文本搜索**: 在所有 Markdown 文件中进行文本匹配
- **相关性评分**: 基于匹配类型和频率计算得分
- **实时索引**: 每次搜索时动态分析文件内容

### 支持的文件类型
- 所有 `.md` 文件
- 自动识别扩展生成的对话文件
- 支持自定义输出目录

### 性能优化
- 异步文件处理
- 内存高效的文本搜索
- 智能的文件过滤
