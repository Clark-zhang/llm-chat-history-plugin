📋 使用说明

  1. 点击上方的 "下载图标为 PNG" 按钮
  2. 图标将自动保存为 icon.png
  3. 将下载的 icon.png 移动到项目根目录：D:\Projects\llm_chat_history\
  4. 在 package.json 中添加： 
    "icon": "icon.png",
  5. 重新打包：vsce package --allow-missing-repository

🎨 设计说明

  • 💜 渐变背景：紫蓝渐变，代表 AI 和科技感
  • 💬 聊天气泡：三个层叠的对话框，象征对话历史
  • 📄 文档线条：顶部气泡内的文档图标，代表 Markdown 导出
  • ✨ 光点装饰：增加现代感和活力

提示：如果自动下载失败，可以右键点击上方图标 → 另存为图片
